# Homebrew Tap: octopusos/octoctl

This repository is a Homebrew tap for installing `octoctl`.

## Install

```bash
brew tap octopusos/octoctl
brew install octoctl
```

## Upgrade

```bash
brew update
brew upgrade octoctl
```

## Notes

- The actual formula lives in `Formula/octoctl.rb`.
- Versions and SHA256 are updated by the `octoctl` release workflow.
